# Kantinku_17
KantinKu membuktikan bahwa kualitas perangkat lunak adalah proses berkelanjutan. Dengan arsitektur yang bersih, pemisahan logika bisnis yang tegas melalui custom hooks dan useReducer, serta pengawasan keamanan yang ketat, proyek ini siap menjadi pondasi sistem layanan kampus yang modern, aman, dan andal bagi mahasiswa.
